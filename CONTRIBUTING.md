# Contributing

Thank you for your interest in contributing to the Atlas-EDC Federation Gateway!

## How to Contribute

1. **Fork** the repository and create a feature branch from `master`.
2. **Make your changes** with clear, descriptive commit messages.
3. **Test** your changes against a running OHDSI WebAPI instance.
4. **Submit a Pull Request** with a description of what you changed and why.

## Reporting Issues

Please use GitHub Issues to report bugs or suggest features. Include:

- Steps to reproduce the issue
- Expected vs actual behavior
- Your environment (Node.js version, OS, EDC connector version if applicable)

## Code Style

- Use clear, descriptive variable names
- Comment complex logic, especially EDC protocol interactions
- Keep functions focused and small
